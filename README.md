# Projeto Kotlin - JetPack Compose

Esse projeto é um aplicativo simples desenvolvido em Kotlin para Android, usando o Jetpack Compose.

A ideia principal foi criar um app com algumas telas e fazer a navegação entre elas. O aplicativo começa em uma tela de login (sem validação), e ao clicar em entrar o usuário é levado para um menu.

No menu, é possível acessar a tela de perfil, a tela de pedidos ou voltar para o login. Cada tela é bem simples, com apenas um título e botões para navegação.

O objetivo do trabalho foi praticar conceitos básicos do desenvolvimento Android, como criação de telas, organização do projeto e navegação entre páginas usando NavController.
